Overview

A matrix, in a mathematical context, is a rectangular array of numbers, symbols, or expressions that are arranged in rows and columns. Matrices are often used in scientific fields such as physics, computer graphics, probability theory, statistics, calculus, numerical analysis, and more.

The dimensions of a matrix, A, are typically denoted as m × n. This means that A has m rows and n columns.

When referring to a specific value in a matrix, called an element, a variable with two indices is often used to denote each element based on its position in the matrix.

For example, given aij, where i = 1 and j = 3, a13 is the value of the element in the first row and the third column of the given matrix.

Matrix operations such as addition, multiplication, subtraction, etc., are similar to what most people are likely accustomed to seeing in basic arithmetic and algebra, but do it often in more ways, and are subject to certain constraints.

Addition

Matrix addition can only be performed on matrices of the same size. This means that you can only add matrices if both matrices are m × n. For example, you can add two or more 3 × 3, 1 × 2, or 4 × 4 matrices. You cannot add a 2 × 3 and a 3 × 2 matrix, a 4 × 4 and a 3 × 3, etc. The number of rows and columns of all matrices must be the same.

If the matrices are the same size, matrix addition is performed by adding the corresponding elements in the matrices. For example, given two matrices, A and B, with elements aij and bij, the matrices are added by adding each element, then placing the result in a new matrix, C, in the corresponding position:

A = | 1 2 3 | B = | 5 6 7 | | 4 5 6 | | 8 9 10 |

In the above matrices:

  • m = 2
  • a11 = 1
  • a12 = 2
  • a13 = 3
  • b11 = 5
  • b12 = 6

We add the corresponding elements to obtain cij:

a11 + b11 = 1 + 5 = 6 = c11

a12 + b12 = 2 + 6 = 8 = c12

a13 + b13 = 3 + 7 = 10 = c13

a21 + b21 = 4 + 8 = 12 = c21

a22 + b22 = 5 + 9 = 14 = c22

a23 + b23 = 6 + 10 = 16 = c23

Thus, matrix C is:

C = | 6 8 10 | | 12 14 16 |

Subtraction

Matrix subtraction is performed in much the same way as matrix addition, with the exception that the values are subtracted rather than added. If necessary, refer to the matrix addition explanation for notation details. Like matrix addition, the matrices being subtracted must be the same size.

If the matrices are the same size, then matrix subtraction is performed by subtracting the elements in the corresponding rows and columns:

A = | 1 2 | B = | 5 6 | | 3 4 | | 7 8 |

a11 - b11 = 1 - 5 = -4 = c11

a12 - b12 = 2 - 6 = -4 = c12

a21 - b21 = 3 - 7 = -4 = c21

a22 - b22 = 4 - 8 = -4 = c22

Thus, matrix C is:

C = | -4 -4 | | -4 -4 |

Multiplication

To multiply two matrices, the number of columns in the first must equal the number of rows in the second. For example, a 2×3 matrix can multiply with a 3×4 matrix, but not with a 4×3. Note: A × B may be valid, but B × A might not.

Matrix multiplication isn’t commutative. It involves taking the dot product of each row from the first matrix with each column from the second. The result is a new matrix with rows from the first and columns from the second.

Example:

A = | 2 4 7 |
| 3 6 1 | B = | 1 2 7 1 |
| 2 4 8 3 |
| 3 1 0 6 |

Since A is 2×3 and B is 3×4, the product C = A × B will be 2×4.

Dot Product Calculations:

c11 = 2×1 + 4×2 + 7×3 = 31

c12 = 2×2 + 4×4 + 7×1 = 27

c13 = 2×7 + 4×8 + 7×0 = 46

c14 = 2×1 + 4×3 + 7×6 = 56

c21 = 3×1 + 6×2 + 1×3 = 18

c22 = 3×2 + 6×4 + 1×1 = 31

c23 = 3×7 + 6×8 + 1×0 = 69

c24 = 3×1 + 6×3 + 1×6 = 27

Scalar Example and definition

A scalar matrix operation is the multiplication of every element within a matrix by a scalar, which is a single real number.

The result is a new matrix with the same dimensions as the original, where each entry is the product of the scalar and the corresponding entry from the original matrix.

How to Perform the Operation

  • Identify the scalar and the matrix: A scalar is a single real number, and a matrix is a grid of numbers.

  • Multiply each element: Take the scalar and multiply it by every single element in the matrix.

  • Keep the dimensions the same: The resulting matrix will have the exact same number of rows and columns as the original matrix.

Example

If you have a matrix:

A = [ [4, 3], [2, 12] ]
k = ½

Then scalar multiplication is:

k × A = ½ × [ [4, 3], [2, 12] ]
       = [ [½×4, ½×3], [½×2, ½×12] ]
       = [ [2, 3⁄2], [1, 6] ]

Transposing Matrices

A scalar matrix operation is the multiplication of every element within a matrix by a scalar, which is a single real number. The result is a new matrix with the same dimensions as the original, where each entry is the product of the scalar and the corresponding entry from the original matrix.

For example, multiplying matrix A by a scalar k means each element aij in A becomes k × aij in the new matrix.

Example: Matrix Transposition

Let's say you have the following matrix A:

A = [ [1, 2, 3],
[4, 5, 6] ]

The first row is [1, 2, 3]. This will become the first column of the transposed matrix:

AT = [ [1],
[2],
[3] ]

The second row is [4, 5, 6]. This will become the second column:

AT = [ [1, 4],
[2, 5],
[3, 6] ]

The transposed matrix, AT, is:

AT = [ [1, 4],
[2, 5],
[3, 6] ]
